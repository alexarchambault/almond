Plotting

almond is not tied to a particular plotting library. It exposes various APIs that allow to interface one's plotting library with almond.

Below are examples of such libraries, and they interface with almond.

vegas

vegas is a plotting library for Scala that relies on vega-lite. It can render plots as HTML sections, which can in turn be displayed inline from almond.

???

plotly-scala

plotly-scala allows to define data structures from plotly.js in Scala, then export those to JSON and generate HTML sections rendering those. It has a module relying on the almond Jupyter API, to initialize plotly.js and push plots to the front-end.
