jvm-repr

almond is compatible with the jvm-repr library. jvm-repr allows to register special renderers for one's own classes. These renderers are tailored to Jupyter. A renderer for a type T provides a Map[String, String] for a value of type T. The keys of this map are MIME types, and the values are a text representation of the value for each MIME type.
